# 三行动画试生产记录

## 目标

基于已人工选定的唯一视觉锚点 `lively_bond_companion_v01`，只试做三条高频动作 strip：

- `idle`：6 帧，常驻待机、轻微呼吸和眨眼。
- `waving`：4 帧，轻触回应、打招呼、收到礼物反馈。
- `review`：6 帧，后续共同学习/专注状态；学习只作为动作状态，不是基础身份。

本阶段不生成完整 9 行 atlas，不接入运行时资源包。

## 锁定特征

- 奶白/浅金短发，头顶小翘发。
- 右侧星形发夹。
- 单侧小耳机。
- 蓝白短外套。
- 胸前暖黄色吊坠。
- 明亮亲近的大眼表情。
- 2.5 头身 Q 版桌宠比例。

## 禁止变化

- 不改成双马尾、青绿色长发、Vocaloid 服装或现有 IP 轮廓。
- 不新增复杂漂浮配件、速度线、舞台偶像元素、麦克风、学习桌、书本堆、任务清单或监督工具。
- 不生成完整 atlas。

## 输出目录

```text
motion_trials/
  idle/raw/
  idle/review/
  waving/raw/
  waving/review/
  review/raw/
  review/review/
```

`raw/` 保存原始生成 strip，`review/` 保存 `192x208` 单帧预览或缩略检查图。
